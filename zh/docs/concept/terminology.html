# 术语

术语 描述
Metric Name 指标名
Tags 每个指标名下面可以加多个 Key/Value 类型的 Tag
Time Series 时间线，Metric Name + Tags 的组合对应一条时间线
Field 一个指标下面可以有多个字段
Data Point 一个字段在相应时间点的数据点
